Configuration

Configure where the frontend sends detection requests. For phone access via tunnels, enter the backend tunnel URL.

Current:

If you're accessing this website from your phone via a tunnel (trycloudflare.com URL), you need to set the backend API URL below.

Steps:

  1. On your phone, open the frontend tunnel URL (trycloudflare.com)
  2. Go to Settings page (this page)
  3. Copy the backend tunnel URL from your laptop Terminal 3
  4. Paste it in the "Backend API URL" field above
  5. Click "Save & Test"

Run on your laptop and check Terminal 3 output:

.\run_mobile_backend_tunnel.ps1

↓ Look for this in output:

https://thereof-math-inns-scored.trycloudflare.com
